Diabetes Prediction Project

2024 - July 2025 | Personal Development Project (Updated: July 06, 2025)

Developed a data analysis and prediction system using Python and Colab to predict diabetes outcomes using the Pima Indians Diabetes Dataset. This project, refined as of July 06, 2025, analyzes health metrics to classify individuals as diabetic (Outcome = 1) or non-diabetic (Outcome = 0), leveraging visualizations and a Decision Tree Classifier.

Key Achievements

  • Designed a Python-based data pipeline to load and preprocess the Pima Indians Diabetes Dataset, improving data quality by handling missing values and outliers in features like Glucose, BloodPressure, and BMI.
  • Implemented a Decision Tree Classifier to predict diabetes outcomes, achieving an accuracy of approximately 0.6924 with all features and identifying Glucose as the best single predictor with a tailored accuracy, refined on July 05, 2025.
  • Created interactive scatter plots and histograms using Matplotlib and Seaborn to visualize relationships between features (e.g., BloodPressure, Pregnancies) and outcomes, with updates for real-time plotting added in July 2025.
  • Deployed the model in a Colab notebook with an enhanced interface for users to explore feature impacts and view predictions, updated at 12:07 AM CET, July 06, 2025.

Technical Details

Data Processing: The pipeline processes the Pima Indians Diabetes Dataset from a CSV file, featuring eight attributes: Pregnancies, Glucose, BloodPressure, SkinThickness, Insulin, BMI, DiabetesPedigreeFunction, and Age. Recent updates include age group categorization (Below 30, Above 30) to improve model performance.

Predictive Model: The Decision Tree Classifier evaluates single features (e.g., Glucose with the highest accuracy) and all features together (0.6924 accuracy with a 70-30 train-test split, random_state=42). Tuning was performed on July 05, 2025, to optimize decision thresholds.

Visualization: Interactive plots include scatter plots of BloodPressure vs. Index for diabetes and non-diabetes cases, histograms of Pregnancies with overlaid Age vs. Pregnancies scatter plots, and feature-wise scatter plots (e.g., Glucose vs. Outcome), updated for real-time data exploration in July 2025.

Report Generation: Generates summary reports in PDF format (conceptual, not coded), detailing accuracy metrics, best features, and visualizations, with a new real-time export feature planned for July 2025 updates.

Technologies: Python Pandas NumPy Matplotlib Seaborn Scikit-learn Colab
