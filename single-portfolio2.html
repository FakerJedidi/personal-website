Tunisian House Prices Prediction

2023 - July 2025 | Personal Development Project (Updated: July , 2024)

Developed a predictive model to estimate house prices in Tunisia using a Kaggle dataset, focusing on exploratory data analysis (EDA), outlier handling, and feature engineering. This project, refined as of July 06, 2025, aims to provide accurate price predictions by analyzing spatial and categorical features, with ongoing work on modeling.

Key Achievements

  • Performed EDA to identify and remove outliers (e.g., `price_tnd` < 0.5e+05, `distance_to_capital` > 600 km) using box plots and scatter plots, improving data quality as of July , 2024.
  • Implemented feature engineering, including `Area_Cat` categorization, `Plus_features` (sum of amenities), and target-encoded `location`, `governorate`, and `city`, with updates on July 25 , 2024.
  • Visualized data distributions, correlations, and a Tunisia map plot using `latt` and `long`, with new box plots for feature importance added in July 2024.
  • Handled missing values by imputing `pieces`, `room`, `bathroom` with area-based means and using K-means for `state`, with plans for predictive modeling underway.

Technical Details

Data Processing: The pipeline processes the Tunisian House Prices dataset, featuring numerical attributes (e.g., `Area`, `distance_to_capital`) and categorical variables (e.g., `location`, `governorate`). Outliers were removed based on realistic thresholds, and `age` was dropped due to high missingness.

Feature Engineering: Created `Area_Cat` (1-5 bins), summed amenities into `Plus_features`, and applied log1p to `price_tnd` for normalization. Skewed features were transformed using Box-Cox with λ=0.15.

Visualization: Includes scatter plots (`distance_to_capital` vs. `price_tnd`), histograms, count plots (`location`, `governorate`), heatmaps, and a map plot, updated for real-time exploration in July 2024.

Modeling: Currently in progress, with plans to implement regression models (e.g., GradientBoostingRegressor) to predict `price_tnd`, targeting an RMSE below 0.1 (log scale), scheduled for July 2024 updates.

Technologies: Python Pandas NumPy Matplotlib Seaborn Scipy Scikit-learn Kaggle
