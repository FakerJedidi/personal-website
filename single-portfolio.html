FPL Game Prediction Points Project

2024 - July 2025 | Personal Development Project (Updated: July 06, 2025)

Developed a predictive model to forecast player points for the Fantasy Premier League (FPL), leveraging historical data and machine learning techniques. This project, refined as of July 06, 2025, assists FPL managers in making informed team selections, optimizing points accumulation, and enhancing strategic gameplay using real-time API data.

Key Achievements

  • Designed and implemented a Python-based machine learning model to predict FPL player points, achieving an 85% accuracy with Random Forest Regressor and an RMSE of approximately 2.75 with GradientBoostingRegressor, refined on July 05, 2025.
  • Collected and processed player statistics (goals, assists, minutes played) and fixture data from the FPL API, automating data cleaning, feature engineering, and real-time updates.
  • Developed interactive visualization tools using Matplotlib and Seaborn, including bar charts, histograms, and top performer plots, with new real-time plotting features added in July 2025.
  • Integrated a team selection algorithm to create optimal lineups within a budget, deployed via a Flask web interface for real-time predictions and downloadable reports, updated at 12:13 AM CET, July 06, 2025.

Technical Details

Prediction Model: The model employs a Random Forest Regressor and a GradientBoostingRegressor to predict player points, trained on features like goals_scored, assists, clean_sheets, minutes, now_cost, and fixture difficulty. Preprocessing includes handling missing values and normalizing statistics, with recent tuning on July 05, 2025.

Interactive Visualization: The tool features line graphs and bar charts of predicted points, histograms of player costs, and a bar plot of top performers by total points, with filters for position and team, enhanced with real-time data in July 2025.

Team Selection: The algorithm adjusts expected points by fixture difficulty (1-5 scale) and position (Goalkeeper, Defender, Midfielder, Forward), selecting a 2-5-5-3 lineup within a 100-unit budget, with updates for dynamic team optimization.

Report Generation: Users can generate weekly prediction reports in PDF format, summarizing top players, predicted points, and team lineups, with a new real-time export feature added on July 05, 2025.

Technologies: Python Scikit-learn Pandas Matplotlib Seaborn Flask API Integration
