Mobile Clock Application for Aurax

2023 | - Institut Supérieur des Études Technologiques de Kélibia

Developed as my Projet End of Studying after three years in the Univerity 'Institut supérieur des études technologiques de Kélibia' program, this project created a mobile clock application for Aurax to manage employee attendance. Completed by June 2023, it offers a secure, user-friendly solution with iterative development techniques learned during my studies.

Key Achievements

  • Designed and implemented a mobile app with license activation, PIN authentication, and pointage options (Arrivée, Pause, Retour de pause, Départ) using Android Studio and React Native.
  • Conducted exploratory data analysis and created UML diagrams (class, sequence, activity) to model the system, applying DSI methodologies.
  • Developed management features for admins and managers, including pointage correction and absence management, with a focus on data security.
  • Built an intuitive interface with language selection, PIN authentication, and pointage screens, tested for reliability and scalability.

Technical Details

Development Process: Utilized an iterative and incremental model, dividing the project into phases for analysis, design, and implementation

Technologies: Android Studio, React Native, Node.js, JavaScript, TypeScript, CSS, HTML, and Visual Paradigm for UML modeling.

Features: Includes activation via a 16-character license code, PIN-based authentication, and four pointage options, with admin/manager dashboards for oversight.

Visualizations: Screenshots of language selection, PIN authentication, and pointage interfaces highlight the user experience, developed during the PFE.

Technologies: Android Studio React Native Node.js JavaScript TypeScript CSS HTML Visual Paradigm
